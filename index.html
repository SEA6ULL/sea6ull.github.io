Installing the SEA6ULL Wizard

Current Repository Version: 1.0

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://sea6ull.github.io
  • Name it "sea6ull" without the quotes
  • Go to "Add-ons"
  • Select "Install from zip file".
  • When it asks for the location, select "sea6ull".
  • For SEA6ULL Wizard, install repository.sea6ullwizard-1.0.zip
  • Go back to "Add-ons" and select "Install from repository"
  • Select "SEA6ULLWizard Repository"
  • Select "Program add-ons" and install "SEA6ULL Wizard"

    • Wizard Source: https://sea6ull.github.io
    Kodi: https://kodi.tv/download

    Build List:
    Frut: Short for Arctic Fuse 2, Real Debrid, Umbrella, Trakt. Premium sources only!

    NOTE: A premium subscription for scraping is required for my builds.

    INSTRUCTIONS:

    1. Download/Install Kodi (Currently my builds only supports v21.x) https://kodi.tv/download
    2. Install and open Kodi
    3. Add 'Media Source' to the file manager (Settings -> File Manager)
    4. Select 'Add source' and type in https://sea6ull.github.io
    5. Name it 'sea6ull'
    6. Press back and click 'Add-ons'
    7. Select Install from zip file
    8. If you are asked to enable 'unknown sources', do as instructed
    9. Select 'Install from zip file'
    10. Select what you named the media source (sea6ull)
    11. Select 'repository.sea6ullwizard-x.x.zip'
    12. After it has installed, select 'Install from repository'
    13. Select 'SEA6ULLWizard Repository' then 'Program add-ons'
    14. Select 'SEA6ULL Wizard' then 'Install'
    15. After it has installed, a dialogue should pop up, I recommend keeping everything on default, so click 'Continue'
    16. Another dialogue will now show, select 'Build Menu' to see the available builds
    17. You will now see the build menu, after you have chosen a build, simply select it and then select 'Fresh Install' and then 'Continue'
    18. The build will now install, once it has finished, Kodi should automatically close, depending on the OS you are using
    19. Simply reopen Kodi, and your build will be installed!

    The first boot may take a few minutes for the widgets to populate, this is normal.
    Your build should now look like one of the following images, if it does not, follow the troubleshooting steps further down this post.

    Activating your premium subscription:
    I will now talk you through on adding your purchased subscription.

    1. Open settings
    2. Click 'Add-ons
    3. Click 'My Add-ons
    4. Click 'Video add-ons'
    5. Click 'Umbrella'
    6. Click 'Configure'
    7. Scroll down to 'Accounts' in the list of options
    8. Now go to the right pane (you can do this by pressing right on your keyboard or remote device.
    9. Scroll down to 'Enable Real-Debrid'
    10. Click 'Authorize Real-Debrid'
    11. You should now be instructed to go to a web-page and insert a code. After doing so, the dialog should automatically disable. Your account is now added and you can return to the home screen and begin watching content!

    Troubleshooting:
    If for some reason, after restarting Kodi post-install, it does not look like one of the attached images, you may need to manually apply the GUI settings file. However, the included wizard makes this job very easy! Instructions are:

    1. Open the settings
    2. Click 'Add-ons
    3. Click 'My Add-ons
    4. Click 'Program add-ons'
    5. Click 'SEA6ULL Wizard'
    6. Click 'Open'
    7. Click 'Builds'
    8. Select the build you installed in the previous step.
    9. Click 'Apply guiFix'
    10. Click 'Apply Fix'
    11. After it has installed, it should force close Kodi, if it does not, it should instruct you to do so, during the next boot the build should be working!

    WARNING: Force close does not work on all OS. If it fails it should tell you. You MUST force close after otherwise, it won't extract the build. Google how to force close an app on your device if you struggle.
